# Airbnb NYC Data Analysis

This project analyzes Airbnb listings in New York City using Python and Power BI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Power BI

## Project Objectives
- Analyze Airbnb listing prices
- Identify patterns in location and room types
- Build machine learning models for price prediction
- Create an interactive Power BI dashboard

## Key Insights
- Manhattan has the highest average Airbnb prices.
- Entire homes account for nearly 50% of listings.
- Listings are concentrated in Manhattan and Brooklyn.
- Lower priced listings tend to receive more reviews.
